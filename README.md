# 📚 Online Bookstore Database (MySQL)

## 📌 Project Overview

This project implements a **relational database for an online bookstore** using MySQL.
It manages information about books, customers, and orders, and demonstrates the use of SQL queries for data retrieval and analysis.
---
## 🗂 Dataset
The dataset contains three CSV files:

* **Books** – Information about books such as title, author, genre, price, and stock.
* **Customers** – Customer details including name and country.
* **Orders** – Records of book purchases made by customers.
---
## 🗄 Database Schema
The database consists of three main tables:
* **Books**
* **Customers**
* **Orders**
Relationship structure:
Customers → Orders → Books
---
## 🧠 SQL Concepts Used
This project demonstrates several SQL concepts:
* Table creation
* Data import from CSV
* Filtering and sorting
* JOIN operations
* Aggregation functions
* GROUP BY and HAVING
* Data analysis queries
---
## 📊 Example Queries
Examples of queries included in this project:
* Retrieve all fiction books
* Find books published after a specific year
* List customers from a particular country
* Calculate total stock of books
* Find total books sold per genre
